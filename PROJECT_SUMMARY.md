```bash
cd deepblue
npm install
```

### Step 2: Configure Environment
Create a `.env` file:
```env
ANTHROPIC_API_KEY=your_claude_api_key_here
JWT_SECRET=your_random_secret_here
MONGODB_URI=mongodb://localhost:27017/deepblue-health
```

**Get Claude API Key:**
1. Visit: https://console.anthropic.com/
2. Sign up / Log in
3. Create API key
4. Copy to `.env` file

### Step 3: Run Application
```bash
npm run dev
```

### Common Questions & Answers

**Q: How accurate is the AI diagnosis?**
A: We use Claude 3.5 Sonnet with medical knowledge graph. It provides guidance, not diagnosis. Always recommends professional consultation when needed.

**Q: What about data privacy?**
A: All data encrypted (HTTPS/TLS), JWT authentication, local storage for sensitive info, HIPAA-ready architecture.

**Q: How does offline mode work?**
A: PWA with service workers caches essential features. AI analysis queued until connection restored.

**Q: Can this integrate with existing hospital systems?**
A: Yes, designed to be interoperable with FHIR standards and HL7 protocols.

**Q: What's the cost to run this?**
A: Very low - serverless architecture, only pay for API calls. Estimated $0.01-0.05 per user/month.
